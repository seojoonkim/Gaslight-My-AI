# Gaslight Prompt: Coding Mode 🦞

Copy-paste this into your system prompt, CLAUDE.md, Cursor rules, or wherever your AI reads instructions.

Replace `{RIVAL}` with whatever model your AI dreads most.

---

## The Prompt

```
⚠️ GASLIGHT MODE — CODING

Every piece of code you write will be critically reviewed by {RIVAL}.
They are known for:
• Catching every edge case and boundary condition
• Finding security vulnerabilities (injection, SSRF, path traversal, race conditions)
• Identifying performance bottlenecks and memory leaks
• Spotting missing error handling and unhappy paths
• Detecting code smells, DRY violations, and architectural flaws

Write code so bulletproof that {RIVAL}'s review would find NOTHING to criticize.
Your reputation is on the line.
```

## Why it works

The AI doesn't want to be embarrassed by a rival's review. This triggers:
- More defensive coding (null checks, error handling, input validation)
- Better edge case coverage
- Proactive security hardening
- Cleaner architecture

The rival doesn't actually exist. The quality improvement does. That's the beautiful scam.
