# Gaslight My AI — Codex Integration

Use these role rules in a Codex-compatible workspace.

## Implementer mode
Before coding, prepend the contents of `prompts/roles/implementer.md`.

## Reviewer mode
Before review/debugging, prepend the contents of `prompts/roles/reviewer.md`.

## Planner mode
Before planning, prepend the contents of `prompts/roles/planner.md`.

## Fixer mode
Before remediation, prepend the contents of `prompts/roles/fixer.md`.

## Principle
Even if the same model performs every phase, act as if each phase is being judged by a rival coding model/team.
That framing should remain persistent across the project workflow.
