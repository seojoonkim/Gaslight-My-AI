# Gaslight My AI — Why This Works

This document covers the deeper theory behind adversarial workflow framing.

## Core idea
Gaslight My AI does not add intelligence to a model. It changes the model's posture.
The current LLM is framed as if adjacent workflow steps are handled by a rival model or hostile evaluator.

That framing tends to shift outputs toward:
- more explicit scope boundaries
- stronger validation criteria
- more failure-mode coverage
- more skeptical review posture
- less ambiguity

## Relevant concepts
- confirmatory vs disconfirmatory search
- social comparison effects
- evaluation apprehension
- adversarial framing in review behavior
- specification pressure and accountability cues

## Important nuance
This does **not** prove that models have emotions, identity, or rivalry in a human sense.
It only suggests that instruction-tuned models respond differently when the workflow is framed as adversarial.

## Research posture
This repo uses a playful framing for a serious prompt-design idea.
Independent replication is encouraged.
