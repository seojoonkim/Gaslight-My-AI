# 🦞 Gaslight My AI — FAQ

## Isn't the name problematic?
"Gaslight" is tongue-in-cheek branding. The actual technique is adversarial workflow framing.
If you need a more corporate-friendly label, call it:
- Adversarial Workflow Context
- Rival-LLM Framing
- Competitive Review Prompting

## Is this just prompt engineering?
Yes — but specifically prompt engineering designed to alter workflow posture, not just wording.

## Does this prove LLMs have feelings?
No. It only shows that framing affects output behavior.

## Can I use this in a team setting?
Yes. In formal environments, use a neutral alias and cite the technique rather than the joke.

## Why micro-chain instead of one big chain?
Because agent-style CLIs often wander. Stage-by-stage artifact handoff is more reliable.
